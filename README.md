🛰️ Network Sentinel: Real-Time Monitoring & Performance Suite
An integrated Python-to-Web solution for automated network discovery and health tracking.

This project bridges the gap between low-level network administration and modern web visualization. By leveraging Scapy for ARP-based scanning and Vanilla JavaScript for dynamic updates, it provides a seamless "single-pane-of-glass" view of every device on your local network.

🌟 Key Technical Highlights
🔍 Automated Asset Discovery: High-speed ARP scanning via Scapy to map local network topology in real-time.

⚡ Performance Analytics: Precise latency tracking (ms) for every active host to identify network bottlenecks.

🔄 Dynamic Data Pipeline: A seamless bridge between a Python 3.11 backend and a frontend dashboard using JSON.

🌙 Responsive Dark-Mode UI: A clean, high-contrast dashboard designed for 24/7 monitoring environments.

🛠️ Tech Stack & Tools
Backend: Python 3.11 (utilizing scapy and subprocess).

Frontend: HTML5, CSS3 (Flexbox/Grid), and Vanilla JavaScript (ES6+).

Driver: Npcap (WinPcap compatible mode) for raw packet access on Windows.
